# NEU Library Visitor Log

A web-based library visitor log system for New Era University.

## Live App
https://neu-library-visitor-log-51899.web.app

## Features
- Google Sign-In restricted to NEU email accounts
- Visitors can log their reason for visiting the library
- Admin dashboard with visitor statistics
- Filter by date, reason for visit
- Search visitors by name or reason
- Block/unblock visitors
- Export visitor log to PDF

## Tech Stack
- React + Vite
- Firebase Authentication
- Firebase Firestore
- Firebase Hosting

## How to Use
- Sign in with your NEU Google account
- Regular users will be greeted and can log their visit
- Admin users will be directed to the admin dashboard

Here is the live app link: https://neu-library-visitor-log-51899.web.app
